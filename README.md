# Student Grade Management System (Java)

## Overview
This project is a Java-based application designed to store and process student grades. It demonstrates fundamental programming concepts including arrays, methods, and control structures to analyze data and produce meaningful results.

## Features
- Accepts multiple student grades as input
- Calculates the average grade
- Identifies the highest grade
- Displays results in a clear format

## Technologies Used
- Java
- Command Line Interface (CLI)

## Skills Demonstrated
- Structured programming
- Arrays and data storage
- Method creation and reuse
- Looping and conditional logic
- Problem-solving and program design

## How It Works
1. The user enters a set of student grades.
2. The program stores the values in an array.
3. It calculates the average of all grades.
4. It determines the highest grade entered.
5. The results are displayed to the user.
